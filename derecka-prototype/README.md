# DERECKA-like Website Prototype

Static HTML/CSS prototype to kick off development.

## Structure
```
derecka-prototype/
├─ index.html
├─ about.html
├─ consortium.html
├─ events.html
├─ publications.html
├─ contact.html
├─ assets/
│  ├─ css/style.css
│  ├─ js/script.js
│  └─ img/ (put images here)
└─ README.md
```

## How to run
Open `index.html` in a browser. No build step is required.

## Suggested next steps
- Add real images to `assets/img/`
- Make it responsive (mobile) in `style.css`
- Replace placeholder texts with real content
- Later: integrate a CMS or backend if needed
